# 产品架构

# 概述

AIP是为了推动数字化转型和创新发展而建设的基础设施系统，通过提供开发工具、测试环境、运行平台和协作工具等功能，帮助企业提高竞争力、降低成本、支持创新和业务拓展，以及提升团队协作效率。

核心框架
核心框架模块包括核心开发框架、前端开发框架（桌面）和前端开发框架（移动）。
  • 核心开发框架
    用于核心开发的框架。 、高性能且经济实惠的应用的最佳实践。
