# 产品架构

# 概述

AIP是为了推动超级自动化转型和创新发展而建设的智能体平台系统，通过提供开发工具、测试环境、运行平台和协作工具等功能，帮助企业提高竞争力、降低成本、支持创新和业务拓展，以及提升团队协作效率。

基础服务
基础服务模块包括权限配置服务、代码生成器、分布式配置服务、事务消息服务、通知管理服务、存储管理服务、单点登录服务、网关配置服务、安全验证码服务、IM消息服务、邮箱管理服务、文档搜索服务、分布式ID服务、敏感词过滤服务、支付服务、会员服务、内容服务、商城服务、流程中心服务和文档查看服务。
  • 权限配置服务
    用于配置权限的服务，管理用户权限和角色权限
  • 代码生成器
    用于生成代码的工具，快速生成项目代码和模块
  • 分布式配置服务
    用于分布式配置的服务，集中管理分布式系统的配置信息
  • 事务消息服务
    用于处理事务消息的服务，保证消息的可靠性和一致性
  • 通知管理服务
    用于管理通知的服务，发送和管理系统通知和消息
  • 存储管理服务
    用于管理存储的服务，提供文件和对象的存储管理功能
  • 单点登录服务
    用于实现单点登录的服务，提供统一的身份认证和授权功能
  • 网关配置服务
    用于配置网关的服务，管理网关的路由和过滤规则
  • 安全验证码服务
    用于生成安全验证码的服务，防止恶意攻击和机器人行为
  • IM消息服务
    用于处理即时消息的服务，实现实时通讯和消息推送功能
  • 邮箱管理服务
    用于管理邮箱的服务，提供邮箱账号和邮件的管理功能
  • 文档搜索服务
    用于搜索文档的服务，提供全文搜索和文档检索功能
  • 分布式ID服务
    用于生成分布式ID的服务，保证ID的唯一性和分布式生成
  • 敏感词过滤服务
    用于过滤敏感词的服务，检测和过滤文本中的敏感词汇
  • 支付服务
    用于处理支付的服务，提供支付接口和支付流程管理
  • 会员服务
    用于管理会员的服务，提供会员注册、登录和会员信息管理
  • 内容服务
    用于管理内容的服务，包括内容发布、内容审核和内容管理
  • 商城服务
    用于管理商城的服务，提供商品管理、订单管理和购物车功能
  • 流程中心服务
    用于流程中心的服务，提供工作流程的定义、执行和监控
  • 文档查看服务
    用于查看文档的服务，支持多种文档格式的在线预览
  • 基设平台管理服务
    用于管理基设平台的服务，提供基设平台的配置和管理功能
数据服务
数据服务模块包括主数据服务、数据上报服务、数据集成服务、数据开发服务、实时计算服务、数据安全服务、数据质量服务、数据总线服务、数据资产服务和数据算法服务。
  • 主数据服务
    用于主数据管理的服务，集中管理和维护组织的核心数据
  • 数据上报服务
    用于数据上报的服务，支持数据的采集、上传和存储
  • 数据集成服务
    用于数据集成的服务，实现不同系统间的数据交换和整合
  • 数据开发服务
    用于数据开发的服务，提供数据处理和ETL流程的开发和调度
  • 实时计算服务
    用于实时计算的服务，支持流式数据的实时处理和计算
  • 数据安全服务
    用于数据安全的服务，保护数据的机密性和完整性
  • 数据质量服务
    用于数据质量的服务，检测和修复数据中的质量问题
  • 数据总线服务
    用于数据总线的服务，实现数据的发布、订阅和传输
  • 数据资产服务
    用于数据资产管理的服务，记录和管理组织的数据资产
  • 数据算法服务
    用于数据算法的服务，提供数据挖掘和机器学习算法
智能服务
智能服务模块包括智能推荐服务、智能搜索服务、智能语音服务、智能图像服务、智能视频服务和智能机器人服务。
  • OCR视觉识别服务
    用于OCR视觉识别的服务，实现文字和图像的识别和提取
  • 自然语言识别服务
    用于自然语言识别的服务，实现文本的语义理解和分析
  • GPT推理服务
    用于GPT推理的服务，实现基于预训练模型的自然语言生成和推理
  • 流媒体识别服务
    用于流媒体识别的服务，实现对音视频流的实时识别和分析
  • 语音助手服务
    提供基础平台运营情况分析，获取运营结果通报，同时形成交互形式
  • 目标检测识别服务
    通过提供通用的API接口，帮助开发人员和应用管理者实现流媒体的物体识别检测功能
运维服务
运维服务模块包括自动化任务服务、分布式日志服务、分布式链路跟踪服务、持续集成服务、容器管理服务和监控预警服务。
  • 自动化任务服务
    用于自动化运维任务的服务，实现自动化执行和调度任务
  • 分布式日志服务
    用于分布式日志管理的服务，集中收集、存储和分析日志数据
  • 分布式链路跟踪服务
    用于分布式链路跟踪的服务，追踪和监控分布式系统中的请求链路
  • 持续集成服务
    用于持续集成的服务，自动化构建、测试和部署软件
  • 容器管理服务
    用于容器管理的服务，管理和编排容器化应用程序
  • 监控预警服务
    用于监控和预警的服务，实时监测系统状态并发送预警通知
业务服务
业务服务模块包括商品秒杀服务、实时推荐服务、实时画像服务和无代码开发服务。
  • 商品秒杀服务
    用于商品秒杀的服务，实现高并发的秒杀功能，输出通用秒杀接口
  • 实时推荐服务
    用于实时推荐的服务，提供实时个性化推荐功能，输出通用接口能力
  • 实时画像服务
    用于实时画像的服务，实时分析用户行为和特征，生成用户画像
  • 无代码开发服务
    用于无代码开发的服务，提供可视化的开发环境和工具，用于做偏向简单的业务类型
运营服务
运维服务模块包括自动化任务服务、分布式日志服务、分布式链路跟踪服务、持续集成服务、容器管理服务和监控预警服务。
  • 基设平台服务
    提供架构排版服务，支持快速搭建和调整系统架构
  • 安全感知服务
    用于安全感知的服务，实时监测和分析系统安全状态
  • 智能运营大脑服务
    用于智能运营大脑的服务，集成智能算法和数据分析，提供运营决策支持
  • 移动终端服务
    用于移动终端的服务，提供移动应用开发和管理功能
