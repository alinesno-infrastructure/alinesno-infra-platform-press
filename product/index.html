# 产品列表

# 概述

平台构建 “感知 - 决策 - 执行” 全链路智能体体系，赋能 AI 自主化与场景落地。通过环境实时感知，将数据转化为可复用的数字资产。

中枢推理层依托大模型管理系统，支持动态决策、自主学习与持续进化，强化目标导向的智能行为。

执行层提供低代码开发工具与 API 集成，无缝衔接业务场景，实现智能体的快速部署与环境适应。核心能力覆盖感知、决策推理、学习、自主性、互动适应及目标导向，为企业打造高效、灵活的智能体解决方案。

智能体产品全景图
全链路智能体平台 推动企业集成各行业场景数据、技术、工艺、开发融合人工智能
智能体中枢推理
基于大模型的智能决策系统，集成自然语言处理、计算机视觉等AI能力，支撑智能体自主推理与策略生成。
  • 智能体管理平台
    提供高性能LLM推理引擎，支持多模态交互与复杂逻辑推理，实现智能体认知决策。
  • OCR视觉识别服务
    高精度图文识别引擎，支持多场景文档解析与结构化数据提取，强化智能体视觉感知。
  • 自然语言识别服务
    NLP语义理解引擎，实现文本情感分析、意图识别与知识抽取，赋能智能体语言交互。
  • 目标检测识别服务
    基于深度学习的物体检测框架，支持视频流实时分析与异常行为识别。
智能体感知能力
构建智能体的环境感知系统，提供多源数据采集、清洗、集成与实时计算能力，支撑智能体对物理世界的全面感知。
  • 主数据服务
    建立企业核心数据资产库，实现数据标准化管理与跨系统共享，保障智能体数据一致性。
  • 数据集成服务
    提供ETL可视化编排工具，实现不同系统间数据的高效交换与价值融合。
  • 数据编排服务
    基于自研的数据编排的工作流引擎，支持复杂数据处理逻辑的快速开发与调度。
  • 实时计算服务
    流式数据处理框架，支持毫秒级延迟的实时分析与决策，赋能智能体动态响应。
  • 数据资产服务
    建立数据资产目录与血缘分析体系，实现数据全生命周期管理与价值挖掘。
智能体业务执行
构建智能体行动支撑体系，提供低代码开发工具与业务系统集成能力，实现决策结果的高效落地。
  • 权限配置服务
    RBAC权限管理系统，支持细粒度的智能体行为控制与安全策略配置。
  • 项目生成器
    可视化代码生成工具，支持智能体功能模块的快速构建与个性化工具和技能的定制。
  • 分布式配置服务
    集中式配置管理平台，支持智能体参数动态更新与多环境统一管控。
  • 网关配置服务
    API网关管理系统，提供智能体服务路由、流量控制与安全防护能力。
  • 敏感词过滤服务
    AI驱动的内容安全引擎，实时检测并过滤违规信息，保障智能体交互合规。
